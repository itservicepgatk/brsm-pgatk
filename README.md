# ПО ОО «БРСМ» | ПГАТК им. А.Е. Клещева 🇧🇾

Добро пожаловать в репозиторий официального сайта первичной организации БРСМ Пинского аграрно-технического колледжа.

## 🎯 О проекте
Этот сайт создан для информирования учащихся о деятельности нашей молодежной организации, предстоящих мероприятиях, акциях и волонтерских движениях. 
Проект участвует в конкурсе **«Войти в IT»** в номинации «Лучший молодежный интернет-ресурс».

## 🚀 Функционал
- **Новости и события:** Актуальная информация о жизни колледжа.
- **Фотогалерея:** Отчеты с мероприятий и акций.
- **Вступление в БРСМ:** Информация для тех, кто хочет присоединиться.
- **Нормативная база:** Устав и символика.

## 🛠 Технологии
